# Chapter 4 — Operators in Python

## Arithmetic Operators

| Operator | Meaning |
|---|---|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |

## Example

```python
a = 10
b = 5

print(a + b)
```

## Comparison Operators

- ==
- !=
- >
- <

## Logical Operators

- and
- or
- not

## Notes

- Operators perform calculations.
- Comparison operators return True or False.
