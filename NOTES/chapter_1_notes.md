# Chapter 1 — Introduction to Python

## What is a Computer?

A computer is an electronic machine that takes input, processes it, and gives output.

## Input → Process → Output

- Input = Data given to computer
- Process = Work done by computer
- Output = Final result

## What is Programming?

Programming means giving instructions to a computer.

## First Python Program

```python
print("Hello World")
```

## Notes

- print() is used to display output.
- Text is written inside quotes.
- Python is beginner friendly.
