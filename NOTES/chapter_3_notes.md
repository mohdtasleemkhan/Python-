# Chapter 3 — Variables & Data Types

## Variable

A variable is a container used to store data.

Example:

```python
name = "Tasleem"
```

## Data Types

| Data Type | Example |
|---|---|
| String | "Hello" |
| Integer | 25 |
| Float | 99.5 |
| Boolean | True |

## input()

Used to take input from user.

Example:

```python
name = input("Enter your name: ")
```

## Notes

- Variables store data.
- type() checks data type.
- input() takes user input.
